Install

To install it's necessary to have an original version of the game. Below is an instructional video on how to install and also some step by step instructions.

Installation video

Steam

1. Click on Library and select ALL GAMES.

2. Look for the greyed out “Star Wars Galaxies” and click on it.

3. Click on install.

4. In the new window that comes up, take note of the install location for the game (the text inside the dropdown menu). In my case, it will be installed under C:\Program Files (x86)\Steam. Once you have done that, press Next.

5. A window will now ask if you want to have a shortcut placed on your desktop. There is no point for this unless you plan on trying to connect to SoE’s servers, which you can’t do anymore.

6. Now the game will start installing. Ignore the next window that comes up with the broken link and press Finish.

7. Once it has finished installing, you can continue on to Part II.

Non-Steam

1. Insert the game disc inside the CD Drive.

2. Click Next on the first welcome screen.

3. Click Next again on the welcome screen.

4. Read the License Agreement fully and select I Agree.

5. On the next screen, take note of the Destination Folder. You will need to know this to install SWG Legends.

6. Select Next after that.

7. Let the game install, inserting the discs it asks for when necessary. Read all the fine print inside your old game manual while you wait and admire the screenshots.

8. Once it has finished installing, you can continue on to the next part.
